# download-server

the download server yo

the site comes preloaded with flight.jpg, youre welcome 

## setup

```bash
npm install
npm run dev
```

place files in `./downloads` (or set `DOWNLOAD_DIR=/path/to/folder`).
add a `pinned.txt` in that folder to show a pinned note (or set `PINNED_FILE=note.txt`).

open `http://localhost:3000`.
